## Project Overview
Plato Bakery is a small, local bakery aiming to improve its online presence through a user-friendly website. The site allows customers to view products, learn about the bakery, explore the gallery, and contact the business. The design reflects a warm, vintage bakery aesthetic and is fully responsive for mobile and desktop users.

## Website Goals and Objectives
- Showcase Plato Bakery’s products and brand identity online.
- Provide clear navigation for users (Home, Menu, About, Gallery, Contact).
- Integrate images and descriptions that highlight the bakery’s offerings.
- Serve as a foundation for future e-commerce functionality.

## Key Features and Functionality
- **Homepage (index.html):** Hero image, navigation bar, welcome text.
- **Menu Page (menu.html):** Images and descriptions of bakery items.
- **About Page (about.html):** Story of the bakery, mission, and values.
- **Gallery Page (gallery.html):** Display of bakery products and atmosphere.
- **Contact Page (contact.html):** Contact details and map.
- **Responsive Design:** Desktop and mobile friendly, including a hamburger menu for mobile navigation.
- **Styling:** Vintage-inspired colors, shadows, and rounded images for a warm bakery feel.
